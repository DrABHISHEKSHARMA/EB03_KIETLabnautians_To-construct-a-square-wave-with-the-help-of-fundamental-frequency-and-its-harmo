Objective :To construct a square wave with the help of fundamental frequency and its harmonic component.


Instructions

  • Read the theory of the experiment completely before performing this experiment

  • The fundamental harmonic fo=Ï‰o/2*pi and amplitude of the respective harmonics will be set by the students.

  • The student will have to observe the change in the waveform for different values of the amplitude of integer harmonics and enter the observation into the table manually.

  • One set of all relative values have been already provided on the screen which is used to generate the periodic square waveform.

  • The student will change all the values with the help of sliders accordingly.

  • The student will have to enter the amplitude values of all the integer harmonics into the table manually.

  • V1 is the voltage value of the fundamental harmonic f0.

  • V1, V2, V3, V4, V5, V6, and V7 are the amplitude values of the integer multiples of the harmonic. For example, V2 is the amplitude of the second harmonic, etc.

  • The student can reset the graph(making a square wave) by using reset button .
